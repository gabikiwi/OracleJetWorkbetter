The Oracle JET WorkBetter application is meant as an example of how to setup a Single-Page-Application(SPA)

It contains examples for working with the ojModule and ojRouting feature for template and paging support. It incorporates responsive design patterns to work at all Oracle JET Media Query sizes for Desktop, Tablet (portrait and landscape), and phone.

Please refer to in code comments for more information. For a full reference on what can be done with the JET framework, please visit the JET Website
